易度开发者中心站点
=========================
本站点实际运行： http://developer.everydo.com

为易度的开发者准备的站点，包括：

- 针对企业应用的PaaS开发环境介绍
- 易度对外的开放API

如何运行这个站点
===========================
本站点使用ZCMS构建，运行方法为：

     python bootstrap.py
     bin/buildout
     bin/pserve development.ini

详细参见： https://github.com/everydo/zcms
